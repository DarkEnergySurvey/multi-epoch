=head1 NAME

dessubmit - submit a DES workflow sequence.

=head1 SYNOPSIS

B<dessubmit> [ B<DESfile> ]

=head1 DESCRIPTION

B<dessubmit> submits a nightly processing workflow to the local Condor system.
The workflow will generally consists of processes that execute both locally and
on remote resources.

=head1 OPTIONS

=over 4

=item -h, -help

Print usage information and exit.

=item -nosubmit

Execute B<dessubmit> as a dry-run without submitting any condor jobs.

=item -v, -version

Display version info and exit.

=back
